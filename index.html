Prework Study Guide

HTML

  • HTML stands for HyperText Markup Language.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML elements work together to create a sturdy "frame".
  • HTML handles structure, content, and the meaning of each element on the page

CSS

  • CSS stands for Cascading Style Sheets.
  • Used to describe the presentation of a website.
  • associated with how each element on a webpage is rendered

Git

  • A Free, open-source version control system.
  • Allows web developers to collaborate on projects.
  • Allows developers to revert to previous versions of work if a problem is discovered.
  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.

JavaScript

  • A lightweight programming language.
  • Used to provide logic and interactivity for websites.
